# ADRs

Append-only log of the decisions behind this project. One file per ADR:
`NNNN-short-title.md`, numbered in order made.

**Rules:** never edit a decided ADR. To reverse one, write a *new* ADR that
supersedes it and flip the old one's status to `Superseded by NNNN`. Keep each to
a screen or less. Current project thinking = `VISION.md` + the sum of these.

Format: **Status · Context · Decision · Alternatives · Consequences.**
